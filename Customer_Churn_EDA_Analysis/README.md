# 📊 Customer Churn Analysis — Exploratory Data Analysis (EDA)


**🔍 Project Overview**

This project focuses on performing Exploratory Data Analysis (EDA) on a telecom customer dataset to uncover key patterns, trends, and factors influencing customer churn.

The objective is to understand customer behavior and identify insights that can support data-driven business decisions and improve customer retention strategies.

**📁 Dataset**

Telecom Customer Churn Dataset
Contains customer demographics, services, billing, and churn status


**⚙️ Tools & Technologies**
Python (Pandas, NumPy)
Data Visualization: Matplotlib, Seaborn
Jupyter Notebook


**🧹 Data Preprocessing**
Handled missing values (e.g., TotalCharges)
Converted data types for accurate analysis
Removed irrelevant columns (e.g., Customer ID)
Prepared clean, analysis-ready dataset


**📊 Key Visualizations**

The following visualizations were created to analyze churn behavior:

* Customer Churn Distribution (Pie Chart)
* Churn by Contract Type
* Churn by Internet Service
* Churn by Payment Method
* Monthly Charges Distribution (KDE Plot)
* Tenure Distribution (Box Plot)
* Correlation Heatmap
  

**📈 Key Insights**
* Customers with month-to-month contracts have significantly higher churn rates
* Fiber optic users show higher churn compared to DSL users
* Customers with higher monthly charges are more likely to churn
* Short-tenure customers are at higher risk of churn
* Strong correlation observed between tenure and total charges


**🎯 Business Impact** 

Identified high-risk customer segments
Helps businesses design targeted retention strategies
Supports pricing and contract optimization decisions
